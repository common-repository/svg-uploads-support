SVG Uploads Support, Allow to use SVG files on wordpress sites easy, Today every websites use a SVG files on their image files, Wordpress SVG support

== Description ==

Vector images are increasingly common on the web. SVGs provide a scalable alternative without loss of quality, sensitive, light and fast than standard images, with the main benefit that they look sharp and clear on any device and, in general, are smaller in file size.

== Installation ==

It is better to install from the wordpress repository, in case you can not install, you can manually download and install it in the "wp-content/plugins".
